# CodeAlpha_Data Visualization
CodeAlpha Internship Task 3: Data Visualization

## Dataset
- Social Media Impact on Health for Exploratory Data Analysis
- Dataset use : https://www.kaggle.com/datasets/sunil123kumar/social-media-impact-on-mental-health
- suggested Target Variables -- (Depression_Label, Stress_level, Social_Media_Hours, Sleep_Hours)

## Columns in Dataset
Rows (1200 Teenagers)
Columns (13 Features)

- Teen_ID -integer
- Age - integer
- Gender - categorical
- Social_Media_Hours - float
- Preferred_Platform - categorical
- Sleep_Hours - float
- Physical_Activity_Level - integer
- Academic_Performance - float
- Stress_Level - integer
- Anxiety_Level - integer
- Depression_Label - binary

## Key Insights & Trends
- Sleep Deprivation and Depression
- Higher Social Media Usage
- Increased Stress and Anxiety
- Physical Activity as a Protective Factor
- Gender-Based Differences
- Academic Performance Shows Weak Correlation

## Tools used
- Python libraries (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
